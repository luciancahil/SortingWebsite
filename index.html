Steps

Explanation

Selection sort works by iterating through each element, and swaping the smallest element with the first element that isn't yet sorted.

We start by Setting the first node as both the "current" and "smallest" Node. We then iterate through each node after it. Every time we find a node smaller than the "smallest" Node, we set that Node as the new "Smallest" Node. After iterating through each Element,we switch the "current" and "smallest" node. Repeat for each Node after the first.
